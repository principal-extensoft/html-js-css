How to Switch Themes

To change the theme in your dashboard, edit the health-dashboard.html file and uncomment the desired theme CSS link:

  • Wall Street Power: Uncomment <link rel="stylesheet" href="styles/theme-wallstreet.css">
  • Federal Government: Uncomment <link rel="stylesheet" href="styles/theme-federal.css">
  • Business Elegance: Uncomment <link rel="stylesheet" href="styles/theme-elegance.css">
  • Early 2000s Glassy: Uncomment <link rel="stylesheet" href="styles/theme-glassy.css">
  • Mainframe Terminal: Uncomment <link rel="stylesheet" href="styles/theme-mainframe.css">
  • Raiders of the Lost Ark: Uncomment <link rel="stylesheet" href="styles/theme-raiders.css">
  • Patriotic: Uncomment <link rel="stylesheet" href="styles/theme-patriotic.css">

Make sure to comment out the other theme links so only one is active.
